## Como rodar o projeto

### 1. Clone o repositório

```bash
git clone https://github.com/seu-usuario/meu-projeto-cassi.git
```

### 2. Instale as dependências

```bash
npm install
```

### 3. Rode o projeto

```bash
npm run dev
```
